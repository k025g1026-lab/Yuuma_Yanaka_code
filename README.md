# ゲーム操作方法

## ゲーム概要
本作は、WASDでプレイヤーを操作し、迫りくる障害物や壁を回避しながら時間を稼ぐサバイバル回避ゲームです。  
特殊能力「リワインド」を活用し、難易度が徐々に上昇する中での高スコア（生存時間）を目指します。

## 基本操作

| キー          | 動作内容                          | 説明 |
|---------------|-----------------------------------|------|
| **W / A / S / D** | プレイヤーの移動 | 上 / 左 / 下 / 右方向へ移動します。 |
| **SPACE**     | リワインドマークの設置 / 発動 | 詳細は下記「リワインド機能」を参照 |
| **R**         | リザルト画面からタイトルへ戻る | ゲーム終了後に使用 |
| **ESC**       | ゲームの強制終了 | いつでも使用可能 |

### プレイヤー移動
- **WASDキー** でプレイヤーを自由に移動できます。
- プレイヤーは画面端に到達すると、それ以上は移動できません（壁で跳ね返るような制限はありません）。

## リワインド機能（特殊能力）

このゲームの最大の特徴です。

1. **マークの設置**  
   - **SPACEキー** を押すと、**現在のプレイヤー位置にリワインドマーク**（旗のアイコン）が設置されます。
   - マークは設置してから**10秒間**有効です。10秒を超えると自動的に消滅します。

2. **リワインドの発動**  
   - マークが設置されている状態で、もう一度 **SPACEキー** を押すと、プレイヤーがマークの位置まで**瞬間移動**（リワインド）します。
   - リワインドした瞬間、移動前の位置に**歪曲フィールド（残像）** が生成され、障害物を引き寄せ・吸収する効果を発揮します。

**ポイント**
- マークは1つしか設置できません（新しいマークを置くと古いものは消えます）。
- 残り時間は画面上のマークの大きさで視覚的に確認できます。
- 難易度が高い場面で、ピンチを脱出する強力な手段です。

## ゲームの流れ

1. **タイトル画面**  
   → **SPACE** で操作説明画面へ

2. **操作説明画面**  
   → **SPACE** でゲーム開始（全データリセット）

3. **ゲーム本編**
   - 時間を生き延びるほどスコア（生存時間）が上昇
   - 壁の出現回数（レベル）が上がると難易度が急激に上昇
   - 障害物に当たる、または壁の隙間を通り抜けられなかった場合 **ゲームオーバー**

4. **リザルト画面**
   - 今回のタイムとハイスコア上位5件を表示
   - **Rキー** でタイトル画面に戻る

## 画面表示

- **上部右側**：現在の生存時間（秒）
- **その下**：壁出現回数（レベル）
- **背景**：星が左へスクロール
- **リワインドマーク**：旗のアイコン（残り時間で縮小）

## Tips（攻略のヒント）

- 早い段階でリワインドマークを設置しておき、危険なタイミングで即座に発動できるようにする。
- 歪曲フィールドは障害物を吸収するだけでなく、合体すると強力になる。
- 壁は左右から交互に出現するため、常に画面中央付近を意識すると安全。
- 後半は壁の隙間が狭くなるため、リワインドを温存しておくことが重要。
